在 Docker 中，如果需要 批量导出所有镜像 并迁移到另一台机器，可以使用以下方法：

📌 方法 1：批量导出所有镜像（推荐）

1. 导出所有镜像

说明：

  • docker images --format "{{.Repository}}:{{.Tag}}" 提取所有镜像的 名称:标签。
  • grep -v "<none>" 过滤掉 <none> 的悬空镜像。
  • docker save $(cat images.list) 批量打包所有镜像到 all_docker_images.tar。

2. 将 all_docker_images.tar 传输到新机器

或使用 U盘、网盘等方式。

3. 在新机器上导入所有镜像

导入后，运行 docker images 检查是否成功。
