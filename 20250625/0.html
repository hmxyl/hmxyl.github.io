全局 Gitignore 文件的位置

全局 Gitignore 文件 (gitignore_global) 的位置取决于你的配置，以下是相关信息：

默认位置

默认情况下，全局 Gitignore 文件通常位于用户主目录下，名为 .gitignore_global：

  • Linux/macOS: ~/.gitignore_global
  • Windows: C:\Users\<你的用户名>\.gitignore_global

注意事项

  • 全局 Gitignore 会应用于你系统上的所有 Git 仓库
  • 项目特定的 .gitignore 会覆盖全局设置
  • 更改全局 Gitignore 不会自动影响已经跟踪的文件，你需要使用 git rm --cached 来停止跟踪这些文件
