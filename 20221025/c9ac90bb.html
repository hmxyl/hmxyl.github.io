UML类图与接口图的表示

发表于 2022-10-25 更新于 2024-09-29 分类于 博客搭建 ， UML类图

类图

①访问修饰符

	+ ：public
	-： private
	# : protected（friendly也归入这类）

② + a : int = defaultValue解读

	a：成员变量名
	int： 变量名的类型
	defaultValue： 为a的默认值

③ + operation1(int params):returnType解读
	operation1： 方法名
	params： 方法参数名
	reternType： 返回值类型
